# Worry Burner 브랜드 가이드라인

> "고민을 태워, 마음을 비우다"

---

## 1. 브랜드 기초

### 브랜드 네임
- **영문**: Worry Burner
- **한글**: 워리버너
- **의미**: 고민(Worry)을 태우는(Burn) 사람/도구

### 태그라인
- **Primary**: 고민을 태워, 마음을 비우다
- **Marketing**: 오늘 밤, 고민을 태워버리세요
- **Short**: 24시간 후, 영원히 사라집니다

### 미션
디지털 공간에서 안전하게 고민을 배출하고, 심리적 해방감을 경험할 수 있는 **따뜻한 피난처**를 제공합니다.

### 핵심 가치
| 가치 | 설명 |
|------|------|
| 🔒 완전한 익명성 | 누가 썼는지 절대 알 수 없음 |
| ⏳ 휘발성 | 24시간 후 영구 삭제 |
| 🤗 따뜻한 위로 | 판단 없이 공감 |
| 🔥 의식적 해방 | 불태우는 의식을 통한 치유 |

## 2. 브랜드 퍼스널리티

### 아키타입
**돌봄이 (Caregiver) + 현자 (Sage)**

### 성격 특성
✅ 따뜻한, 신뢰할 수 있는, 차분한, 감성적인, 미니멀한

❌ 시끄러운, 화려한, 장난스러운, 기업적인, 차가운

### 브랜드 페르소나: "불멍이"
- 20대 후반, 조용히 곁에서 들어주는 친구
- 말은 적지만 필요할 때 따뜻한 한마디

## 3. 로고

### 구성
```
🔥 Worry Burner
```

### 변형
| 유형 | 형식 | 용도 |
|------|------|------|
| Full | 🔥 Worry Burner | 메인 페이지 |
| Stacked | 🔥 + 줄바꿈 + Worry Burner | 세로형 |
| Symbol | 🔥 | 파비콘 |
| Text Only | Worry Burner | 복잡한 배경 |

### 사용 규칙
- **최소 크기**: 24px height
- **여백**: 로고 높이의 50%
- **배경**: campfire-900 (#1C1917) 권장

### ❌ 금지 사항
- 색상 변경
- 비율 변형
- 그림자/효과 추가
- 폰트 변경

## 4. 톤 앤 보이스

### 전체 톤
**따뜻하고 차분한** + **친근하고 위로하는**

### 작문 원칙

| 원칙 | ✅ Good | ❌ Bad |
|------|---------|--------|
| 공감 먼저 | 힘들었겠다 | 왜 그런 생각을 했어? |
| 짧고 따뜻하게 | 다 지나갈 거야 🌙 | 인생은 원래 어려운... |
| 부드러운 제안 | 여기에 적어봐 | 고민을 입력하세요 |
| 이모지 절제 | 푹 자 🌙 | 푹 자 🌙✨💫🌟 |

### UI 카피 예시
```
placeholder: "여기에 고민을 적어주세요... 🔥"
button: "태워버리기 🔥"
loading: "불을 피우는 중..."
success: "제 고민이 한 줌의 재가 되었습니다"
```

### 용어 사전
| ✅ 사용 | ❌ 미사용 |
|--------|----------|
| 고민 | 걱정, 생각 |
| 태우다 | 삭제하다 |
| 토닥토닥 | 좋아요 |
| 엿보기 | 둘러보기 |
